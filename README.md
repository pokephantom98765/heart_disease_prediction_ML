# Heart Disease Prediction using Machine Learning

A Machine Learning project that predicts whether a person is likely to have heart disease based on medical parameters and health data.

---

## Overview

This project uses machine learning algorithms to analyze medical data and predict the presence of heart disease. The system helps in understanding how predictive analytics can assist in healthcare applications.

---

## Features

- Heart disease prediction system
- Medical data analysis
- Data preprocessing
- Machine Learning model training
- Prediction and accuracy evaluation

---

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

---

## Dataset Features

The dataset includes medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Slope
- Number of Major Vessels
- Thalassemia

---

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Data Visualization
4. Train-Test Split
5. Model Training
6. Prediction System
7. Accuracy Evaluation

---

## Model Used

- Logistic Regression

---

## Results

The model predicts the possibility of heart disease with good accuracy using patient medical data.
